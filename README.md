About the project:

League of legends app is a project that will help you learn basic information about the heroes of league of legends. Information are fetched using an external API from RIOT GAMES.

Build with:<br/>
» React JS<br/>
» TypeScript<br/>
» CSS<br/>
» RIOT GAMES API<br/>
